Basic HTML structure

This is H2 Heading

This is a paragraph that describes the basic structure of an HTML document. It includes a DOCTYPE declaration, html, head, and body tags.

HTML documents start with a DOCTYPE declaration, followed by the html tag that contains the head and body sections. The head section includes metadata such as character set and viewport settings, while the body section contains the content that is displayed in the browser.

Subheading Example

Subheadings help organize content within the body of the document. They provide a way to break down information into manageable sections, making it easier to read and understand.

Using headings and paragraphs effectively can enhance the readability of your HTML documents, allowing users to quickly find the information they need.

Conclusion

In conclusion, understanding the basic structure of an HTML document is essential for web development. It sets the foundation for creating well-structured and accessible web pages.

By using appropriate headings and paragraphs, you can create a clear and organized layout that improves user experience.
